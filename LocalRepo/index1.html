git add file name -> adds new or changed files in your working directory to the git staging area

git commit -m "some message" -> it is the record of changes we made in git add

git status -> to show tracked or untracked file

git add . -> it is to include changes in my working directory for the next commit

git commit -m "add files" -> is used to create a new commit in your Git repository with the message "add files"

git push origin main -> upload local repo(repository) to remote(github) repo

git init -> it is used to create a new Git repository. It initializes a new repository in the current directory

Running 'git init' creates a hidden directory named .git in your project folder
